# Executive Summary

## Objective

This project analyzes customer churn for a telecom business. The goal is to identify which customer groups are leaving, how much monthly revenue is at risk, and which retention actions should be prioritized.

## Analysis Plan

1. Clean customer-level churn data.
2. Create churn flag, tenure band, and revenue-at-risk metrics.
3. Analyze churn by contract, tenure, internet service, and payment method.
4. Build Tableau dashboards for churn overview and segment-level retention planning.

## Expected Business Insight

The project should help management separate high-volume customer groups from high-risk groups. The strongest recommendations should focus on customers with high churn rate and high monthly charges.
